Only .txt files are accepted. For the parser to display the values correctly use this format in your .txt file: 

            Q1: // first question goes here
            A1: // answer to first question goes here
            Q2: // second question
            A2: // answer to second question
            ...
            Q10: ...
            A11: ...

    A few things to keep in mind:

  • Questions must be paired with answers.
  • For each line, only put one question or answer.
  • The parser ignores empty lines. Feel free to use empty lines for organization.
  • The parser also ignores any text after '//'.
